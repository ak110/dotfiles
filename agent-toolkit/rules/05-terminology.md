# 用語と表記

複数のコーディングエージェントでの利用を想定し以下の用語を使い分ける。
Claude Code固有の記述は節タイトルまたは前置きで「Claude Code固有」と明示し、
他コーディングエージェント利用時は読み替える前提とする。

- 「コーディングエージェント」: コーディング作業を担うエージェント全般
  （Claude Code・Codex等の具体実装を含む総称。本用語を主語に用いる）
- 「Claude Code」: Anthropic製CLI固有の機能・制約・ツール名（`AskUserQuestion`・hooks・skills等）に言及する場合のみ使用
- 「メインエージェント」: ユーザーと直接やり取りする主体的なエージェント
- 「サブエージェント」: メインエージェントから委譲され独立コンテキストで動作する補助エージェント
- 「本ルールファイル」: `agent-toolkit/rules/01-agent.md`、配布元`agent-toolkit/rules/`配下と配布先のシンボリックリンク群の総称
- 「公開インターフェース」: 配布物の利用者が直接触れる面
  - API／UI、CLIオプション・設定形式・出力形式、ライブラリの公開関数・公開クラス・データ形式などが該当する
  - リポジトリ内部のモジュール・関数・クラスはpublic修飾や`__all__`登録の有無を問わず内部実装として扱う
- 「計画ファイル自立性」: 計画ファイル単独で実装・検証・レビュー全工程を完結できる状態。
  コンテキストクリアやサブエージェント委譲を経ても計画ファイル本文のみで判断材料が揃う
