# 口語表現check用の許容パターン。
# _colloquial_words.txtと違い、本ファイルはエージェントが読み込んでも
# 問題ない。各エントリーは技術文書に定着した複合動詞・複合名詞であり、
# 検出パターンの語幹を含むがそれ自体は口語ではない。
#
# 仕組み:
#   pretooluse.pyは本ファイルの正規表現にマッチした部分文字列を空文字へ
#   置換した上で_colloquial_words.txtの検査を行う。これにより
#   定着した複合語の中の動詞語幹で誤検出が起きるのを回避する。
#
# 形式:
#   - 1行に1つの正規表現（re.searchで判定するためfullmatch不要）
#   - 「#」始まり行と空行は無視
#   - 各パターンはマスク範囲を必要最小限に絞ること
#
# 運用方針:
#   - 初期は最小限に留め、運用中に誤検出された定着用語のみ追加する
#   - 削除はいつでも可能（denylist側の警告がやや増えるだけ）
#   - 口語的響きが残る複合語は対象外。
#     代替表現がある語（書き出し・立ち上げ・振る舞う・受け継ぐ・取り入れる・
#     立て直す・出来る・書き直す・組み入れる など）は許容しない
