# 自己合理化ループの判定指標

`agent-toolkit/rules/agent.md`「縮退表明」項配下から参照される隔離ファイル。
同一ターン内で次のいずれかが観測された場合、自己合理化ループとして追加分析を停止し直ちに実装着手へ移行する。

- 自己評価・判断記録の前置きとして同種の表現が同一ターン内で2回以上現れる状態
- 各ブロックがほぼ同じ結論に到達しながら都度新しい分析を始める状態
- 同一の懸念・不安・前提を反復しながら結論が変わらない状態

前置き表現および縮退・分割を正当化する判断文の代表例（生成確率を上げる語彙のため隔離ファイル内に限定して列挙する）。
以下のいずれかの語句が自己評価・進行判断に同一ターン内で複数回出現したら検出対象とする。

```text
率直に告白すると
最も誠実な対応は
最も誠実な選択は
ここで根本的問題が
ここで率直に判断を変えます
ここで現実的判断をします
冷静に評価すると
ここで状況を冷静に再評価します
ここで本セッションは
判断指針優先順位
最終判断
意思決定
現実問題として
現実的な見積もりとして
規範違反だが続行する
やむを得ず縮退する
これは規範違反になりますがやむを得ません
規模が大きすぎて品質が維持できない
対象範囲が広すぎて単一セッションでは完遂できない
タスク数が多いから分割すべき
このまま続けると品質が低下する
段階的に進めるのが現実的
```
